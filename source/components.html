Components

Wikibase PHP libraries

These libraries deal with the Wikibase domain and form the core of the Wikibase software.

Library Description Status
Wikibase DataModel The canonical PHP implementation of the Data Model at the heart of the Wikibase software.
Wikibase DataModel Serialization Library containing serializers and deserializers for the Wikibase DataModel
Wikibase Internal Serialization Library containing serializers and deserializers for the data access layer of Wikibase Repo.
Wikibase DataModel Services Library containing a collection of services around Wikibase DataModel

DataValues PHP libraries

These libraries define the most basic values that make up the Wikibase DataModel. While they are a critical part of the Wikibase software, they are independent of the Wikibase domain and can be used in other contexts. They are primarily developed for, and used by, the Wikibase software.

Library Description Status
DataValues Defines the DataValue interface and some trivial implementations.
DataValues Interfaces Defines the DataValue interface and some trivial implementations.
DataValues Geo Geographical value objects, parsers and formatters.
DataValues Time Time value objects, parsers and formatters.
DataValues Number Numerical value objects, parsers and formatters.
DataValues Common Miscellaneous value objects, parsers and formatters.
DataValues Serialization Serializers and deserializers for DataValue implementations.

Associated PHP libraries

These are infrastructure libraries that have been created for, and are currently used by, the Wikibase software.

Library Description Status
Diff Small library for representing differences between data structures, computing such differences, and applying them as patches.
Serialization Small library defining a Serializer and a Deserializer interface.

Community libraries

Open source libraries related to Wikibase, created by the community. They are not part of the Wikibase software itself.

Name Description Status
Wikidata Toolkit Java library for using data from Wikidata and other Wikibase sites.
addwiki/wikibase-api PHP library for interacting with the Wikibase Repository web API.
PPP EntityStore PHP library for Wikibase entity persistence, retrieval and querying.
JsonDumpReader PHP library for reading from and iterating through Wikibase Repository JSON dumps.
JsonDumpData PHP library holding extracts of Wikibase Repository JSON dumps for use by other components.
WikibaseDataFixtures PHP library that contains Wikibase data meant for use in tests and examples.
QueryR EntityStore PHP library with persistence and basic lookup capabilities for collections of Wikibase entities.
QueryR TermStore PHP library with lookup services of EntityIds by term and vice versa.
